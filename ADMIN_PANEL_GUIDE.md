# Admin Panel Guide - Optimus Global

This document outlines the available user roles and the core features of the Optimus Global Admin Panel.

## ğŸ‘¥ User Roles

The system supports four distinct roles to maintain security and operational Efficiency:

1.  **Admin (Super User)**
    *   Full access to all modules and settings.
    *   Can manage other users and system configurations.
    *   Unauthorized to be created via standard user routes (restricted to server-side creation).

2.  **Moderator**
    *   Can review and manage student applications and counseling inquiries.
    *   Access to update testimonials and partner details.
    *   Restricted from user management and system settings.

3.  **Content Writer**
    *   Primary access to the **Blogs** and **Media Library**.
    *   Can create, edit, and manage status (Draft/Published) of articles.
    *   Restricted from accessing student applications or financial data.

4.  **User (Limited Access)**
    *   Basic level access for viewing specific dashboard data.
    *   Specific permissions can be assigned based on departmental needs.
